# LawSchoolNotes

Claude Code context for this repository.

## Project
- Vue 3 + Vite app for organizing law school notes, outlines, and case briefs.

## Commands
- Install: `npm install`
- Dev server: `npm run dev`
- Build: `npm run build`
- Unit tests: `npm run test:unit`

## Notes
- Main app source is in `src/`.
- Unit tests are in `src/__tests__/`.
- Keep changes focused and minimal.
